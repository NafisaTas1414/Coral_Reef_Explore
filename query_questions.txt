1. Which coral species are winners vs losers — which grew in abundance and which declined the most? (Nafisa)

Input:

db.occurrences.aggregate([
  { $match: { occurrenceStatus: "present", organismQuantity: { $gt: 0 } }},
  // Sum abundance per species per year
  { $group: {
      _id: { species: "$scientificName", year: { $year: "$eventDate" } },
      totalAbundance: { $sum: "$organismQuantity" }
  }},
  { $sort: { "_id.year": 1 }},
  // Get first and last year totals per species
  { $group: {
      _id: "$_id.species",
      firstAbundance: { $first: "$totalAbundance" },
      lastAbundance:  { $last: "$totalAbundance" }
  }},
  // Calculate change — positive = winner, negative = loser
  { $project: {
      species: "$_id",
      firstAbundance: 1, lastAbundance: 1,
      change: { $subtract: ["$lastAbundance", "$firstAbundance"] }
  }},
  { $sort: { change: -1 }},  // flip to 1 for biggest losers
  { $limit: 10 }
])

3. Steepest decline by region — compare each region's first survey vs most recent survey (Nafisa)

Input:

db.occurrences.aggregate([
  { $match: { occurrenceStatus: "present" }},
  // Get distinct species per region per year
  { $group: {
      _id: { region: "$locality", year: { $year: "$eventDate" } },
      distinctSpecies: { $addToSet: "$scientificName" }
  }},
  { $project: {
      region: "$_id.region",
      year: "$_id.year",
      speciesCount: { $size: "$distinctSpecies" }
  }},
  { $sort: { year: 1 }},
  // For each region grab first and last survey year counts
  { $group: {
      _id: "$region",
      firstYear: { $first: "$year" },
      firstCount: { $first: "$speciesCount" },
      lastYear: { $last: "$year" },
      lastCount: { $last: "$speciesCount" }
  }},
  // Calculate the change (negative = decline)
  { $project: {
      region: "$_id",
      firstYear: 1, firstCount: 1,
      lastYear: 1,  lastCount: 1,
      change: { $subtract: ["$lastCount", "$firstCount"] }
  }},
  { $sort: { change: 1 }}  // most declined first
])

  { region: 'St. Croix, US Virgin Islands',              firstYear: 2015, firstCount: 42, lastYear: 2023, lastCount: 36, change: -6 }
  { region: 'Puerto Rico',                               firstYear: 2014, firstCount: 44, lastYear: 2023, lastCount: 41, change: -3 }
  { region: 'Dry Tortugas, Florida',                     firstYear: 2014, firstCount: 35, lastYear: 2022, lastCount: 32, change: -3 }
  { region: 'St. Thomas and St. John, US Virgin Islands',firstYear: 2013, firstCount: 43, lastYear: 2023, lastCount: 43, change:  0 }
  { region: 'Flower Garden Banks, Gulf of Mexico',       firstYear: 2013, firstCount: 19, lastYear: 2024, lastCount: 19, change:  0 }
  { region: 'Southeast Florida',                         firstYear: 2014, firstCount: 24, lastYear: 2022, lastCount: 26, change: +2 }
  { region: 'Florida Keys, Florida',                     firstYear: 2014, firstCount: 34, lastYear: 2022, lastCount: 41, change: +7 }

Disclaimers:
  - Each region is compared on its own first vs last survey year, not the same calendar years
  - St. Croix saw the steepest decline (-6 species), followed by Puerto Rico and Dry Tortugas (-3 each)
  - Florida Keys actually gained 7 species — suggesting recovery or improved survey coverage
  - Flower Garden Banks last survey is 2024 which is incomplete, so change: 0 may not be accurate

9. Impact of the 3rd Global Coral Bleaching Event (2014-2017) — did species richness and total abundance drop during the bleaching period compared to before and after? (Nafisa)

Context: NOAA confirmed the Third Global Bleaching Event ran from 2014-2017, driven by rising ocean temperatures. This was the longest and most widespread bleaching event on record at the time.

  Periods:
  - Before:  2013          (baseline)
  - During:  2014 – 2017   (bleaching event)
  - After:   2018 – 2019   (recovery window)

Input:

db.occurrences.aggregate([
  { $match: { occurrenceStatus: "present",
      eventDate: { $gte: ISODate("2013-01-01"), $lte: ISODate("2019-12-31") }
  }},
  { $group: {
      _id: { $year: "$eventDate" },
      distinctSpecies: { $addToSet: "$scientificName" },
      totalAbundance: { $sum: "$organismQuantity" }
  }},
  { $project: {
      year: "$_id",
      speciesCount: { $size: "$distinctSpecies" },
      totalAbundance: 1
  }},
  { $sort: { year: 1 }}
])

Output:

  { year: 2013, speciesCount: 44, totalAbundance: 10448 }   baseline
  { year: 2014, speciesCount: 49, totalAbundance: 18149 }   bleaching begins
  { year: 2015, speciesCount: 51, totalAbundance: 17805 }
  { year: 2016, speciesCount: 45, totalAbundance: 11796 }   mid-event drop
  { year: 2017, speciesCount: 50, totalAbundance: 18219 }
  { year: 2018, speciesCount: 43, totalAbundance: 12193 }   post-bleaching
  { year: 2019, speciesCount: 49, totalAbundance: 20899 }  ←partial recovery

Disclaimers:
  - Not all regions are surveyed every year so raw numbers vary
  - 2016 shows the clearest drop in both species count and abundance during the bleaching window
  - Abundance in 2018 is lower than baseline 2013 — suggesting incomplete recovery
